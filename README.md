# 🃏 YGO Database (PT-BR & PT-PT)
[![Netlify Status](https://api.netlify.com/api/v1/badges/your-id-here/deploy-status)](https://ygodatabase.netlify.app/)
[![Vue.js](https://img.shields.io/badge/Vue.js-35495E?style=flat&logo=vuedotjs&logoColor=4FC08D)](https://vuejs.org/)
[![Vuetify](https://img.shields.io/badge/Vuetify-1867C0?style=flat&logo=vuetify&logoColor=white)](https://vuetifyjs.com/)
[![API](https://img.shields.io/badge/YGOPRODeck-API-orange)](https://ygoprodeck.com/api-guide/)

**YGO Database** é um catálogo visual moderno para duelistas e estrategistas de **Yu-Gi-Oh!**. Acesse milhares de cartas atualizadas, filtradas e catalogadas perfeitamente. Construído como um projeto robusto para aplicar arquitetura escalável e conhecimentos em **Vue.js**, **Vuetify** e consumo profundo de **APIs RESTful**.

🌐 **[Acesse o projeto YGODatabase Online aqui!](https://ygodatabase.netlify.app/)**

---

## ✨ Principais Funcionalidades

- 🔍 **Busca Inteligente & Preview**: Busque cartas por nome e receba prévias das artes e descrições dinamicamente através de um *Autocomplete dropdown* com debouncing nativo.
- ⚙️ **Filtros Avançados Universais**: Filtre todo o compêndio ativamente por **Tipos**, **Arquétipos**, **Classes de Monstro** e **Atributos Elementares**.
- 📱 **Responsividade Total**: Fluxo projetado sob o padrão *Mobile First*, contando com _Navigation Drawers_ fluídos, grids flexíveis e layouts ajustáveis.
- 🧱 **Atomic Design**: Componentização guiada pelos preceitos de Design Atômico (Componentização dividida em Átomos, Moléculas), escalável e totalmente manutenível.
- 🌙 **Modern UX/UI**: Feedback visual limpo, loaders elegantes customizados com scrollbars de marca personalizada para entregar uma imersão superior.
- 🇧🇷🇹🇱 **Localização**: Focado totalmente em trazer descrições e bases completas no idioma **Português (Brasil & Portugal)**.

---

## 🛠 Tecnologias Utilizadas

Este projeto foi forjado utilizando ferramentas modernas focadas no front-end web:

* **[Vue.js (v2)](https://vuejs.org/)** - O framework reativo ditando toda a integração interativa.
* **[Vuetify](https://vuetifyjs.com/)** - UI Library robusta, oferecendo componentes baseados no rigor do Material Design.
* **[YGOPRODeck API](https://ygoprodeck.com/api-guide/)** - Fonte inteligente de dados em tempo real (+12,000 cards catalogados).
* **Grid System & CSS3** - Personalizações estritas e transições visuais CSS feitas do zero.

## 🚀 Como Configurar e Executar Localmente

Certifique-se de que você possui o [Node.js](https://nodejs.org/) instalado em seu sistema para iniciar o ambiente.

**1. Faça o clone do repositório em sua máquina:**
```bash
git clone https://github.com/PriseTheSun/YGO-Database.git
cd YGO-Database
```

**2. Instale todas as dependências:**
```bash
npm install
```

**3. Execute o script de ambiente de desenvolvimento:**
```bash
npm run serve
```

> 🌟 Seu servidor local rodará, geralmente na porta `http://localhost:8080/`. O Hot-reload atuará para processar modificações em tempo real.

**4. Para criar uma build minificada visando Produção:**
```bash
npm run build
```
